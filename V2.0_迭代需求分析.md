## 1. 迭代背景与目标

### 1.1 当前状态 (V1.0)
- ✅ 核心拍照识字功能
- ✅ 4级字库系统（100-1500字）
- ✅ 5种音色选择
- ✅ 基础设置管理
- ❌ 无用户体系
- ❌ 数据无法云端同步
- ❌ 无学习进度追踪
- ❌ 跨设备无法使用

### 1.2 V2.0 核心目标
1. **建立用户体系**：家长账户 + 儿童档案
2. **数据持久化**：学习记录云端存储
3. **功能扩展准备**：为绘本、游戏等功能预留架构
4. **合规性**：满足儿童隐私保护法规要求

### 1.3 未来规划 (V3.0+)
- 拍绘本AI讲解
- 个性化识字游戏
- 家长管理后台

## 2. 用户体系设计

### 2.1 账户架构

```
家长账户 (Parent Account)
├── 账户信息
│   ├── 手机号/邮箱（登录凭证）
│   ├── 密码/验证码登录
│   └── 注册时间、最后登录时间
├── 家长设置
│   ├── 使用时长限制（每日/每次）
│   ├── 内容过滤级别
│   └── 通知偏好
├── 付费信息（预留）
│   ├── 会员等级（免费/基础/高级）
│   ├── 订阅状态
│   └── 支付记录
└── 儿童档案列表 (1-N)
    ├── 档案1: 大宝 (3岁)
    ├── 档案2: 二宝 (5岁)
    └── ...

儿童档案 (Child Profile)
├── 基本信息
│   ├── 昵称（必填）
│   ├── 头像（可选，默认卡通头像）
│   ├── 年龄段（3-4/4-5/5-6岁）
│   └── 性别（可选）
├── 学习数据
│   ├── 已学汉字列表及掌握程度
│   ├── 学习时长统计
│   ├── 活跃记录
│   └── 薄弱字记录
├── 个性化设置
│   ├── 偏好音色
│   ├── 当前字库
│   └── 界面主题（预留）
└── 成就数据
    ├── 连续学习天数
    ├── 解锁徽章
    └── 收集进度
```

### 2.3 隐私合规设计

#### 2.3.1 COPPA/GDPR-K 合规要点

| 法规要求 | 实施方案 |
|----------|----------|
| **家长可验证同意** | 注册时要求家长阅读隐私政策并勾选同意；可选手机验证确认成年人身份 |
| **最小化数据收集** | 儿童仅需昵称（可选）和年龄段；不收集真实姓名、地址、学校等 |
| **数据透明** | 家长后台可查看所有收集的数据；明确说明数据用途 |
| **家长权利** | 提供"查看数据"、"导出数据"、"删除账户"功能 |
| **安全措施** | 儿童数据加密存储；图片上传后处理完立即删除 |

#### 2.3.2 隐私政策要点

```
《zizi AI识字伴侣隐私政策》- 家长版

1. 我们收集什么信息？
   - 家长：手机号（用于登录和身份验证）
   - 儿童：昵称（可选）、年龄段（用于内容适配）
   - 学习数据：认识的字、学习时长（用于进度追踪）

2. 我们如何使用信息？
   - 提供个性化学习内容
   - 生成学习报告
   - 改进产品体验

3. 信息共享
   - 不与第三方共享儿童个人信息
   - 学习数据匿名化后用于算法优化

4. 家长权利
   - 随时查看孩子的数据
   - 导出或删除数据
   - 注销账户

5. 联系我们
   - 隐私问题反馈邮箱
```

### 3.2 数据表设计

#### 3.2.1 家长表 (parents)

```sql
CREATE TABLE parents (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    phone           VARCHAR(20) UNIQUE,         -- 手机号（中国大陆）
    email           VARCHAR(255) UNIQUE,        -- 邮箱（国际用户）
    password_hash   VARCHAR(255),               -- 密码哈希（可选）
    auth_provider   VARCHAR(20) DEFAULT 'phone',-- 认证方式：phone/wechat
    auth_id         VARCHAR(100),               -- 第三方认证ID

    -- 家长设置
    settings        JSON,                       -- 通知偏好、隐私设置等

    -- 时间戳
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    last_login_at   TIMESTAMP NULL,

    -- 状态
    status          TINYINT DEFAULT 1,          -- 0:禁用 1:正常

    -- 合规
    accepted_tos_at TIMESTAMP NULL,             -- 同意服务条款时间
    accepted_privacy_at TIMESTAMP NULL          -- 同意隐私政策时间
);
```

#### 3.2.2 儿童档案表 (child_profiles)

```sql
CREATE TABLE child_profiles (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    parent_id       BIGINT NOT NULL,

    -- 基本信息（最小化）
    nickname        VARCHAR(50) NOT NULL,       -- 昵称
    avatar_type     TINYINT DEFAULT 1,          -- 1-10预设卡通头像
    age_group       TINYINT NOT NULL,           -- 1:3-4岁 2:4-5岁 3:5-6岁

    -- 学习设置
    preferred_voice VARCHAR(50) DEFAULT 'serena',-- 偏好音色
    current_library VARCHAR(50) DEFAULT 'infant',-- 当前字库

    -- 统计
    total_chars_learned INT DEFAULT 0,          -- 累计识字数
    total_study_minutes INT DEFAULT 0,          -- 累计学习分钟
    streak_days     INT DEFAULT 0,              -- 连续学习天数
    last_study_at   TIMESTAMP NULL,             -- 最后学习时间

    -- 设置
    settings        JSON,                       -- 个性化设置

    -- 时间戳
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,

    FOREIGN KEY (parent_id) REFERENCES parents(id) ON DELETE CASCADE
);
```

#### 3.2.3 学习记录表 (learning_records)

```sql
CREATE TABLE learning_records (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    child_id        BIGINT NOT NULL,

    -- 学习内容
    char            VARCHAR(10) NOT NULL,       -- 学习的汉字
    library_id      VARCHAR(50),                -- 所属字库

    -- 行为类型
    action_type     VARCHAR(50) NOT NULL,       --
                                              -- 'photo_capture':拍照
                                              -- 'char_viewed':查看汉字
                                              -- 'story_played':播放故事
                                              -- 'char_replayed':重播读音
                                              -- 'quiz_correct':答题正确
                                              -- 'quiz_wrong':答题错误

    -- 上下文信息
    context         JSON,                       -- 扩展信息：
                                              -- photo_url: 照片URL
                                              -- story_text: 故事内容
                                              -- voice_id: 使用的音色
                                              -- source: 来源页面

    -- 时长统计
    duration_sec    INT DEFAULT 0,              -- 停留/学习时长（秒）

    -- 元数据
    device_type     VARCHAR(50),                -- 设备类型
    app_version     VARCHAR(20),                -- App版本

    -- 时间戳
    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    INDEX idx_child_time (child_id, created_at),
    INDEX idx_char (child_id, char),
    FOREIGN KEY (child_id) REFERENCES child_profiles(id) ON DELETE CASCADE
);
```

#### 3.2.4 汉字掌握度表 (char_mastery)

```sql
CREATE TABLE char_mastery (
    id              BIGINT PRIMARY KEY AUTO_INCREMENT,
    child_id        BIGINT NOT NULL,
    char            VARCHAR(10) NOT NULL,       -- 汉字

    -- 掌握程度
    mastery_level   TINYINT DEFAULT 0,          -- 0:初学 1:了解 2:熟悉 3:掌握

    -- 学习统计
    view_count      INT DEFAULT 0,              -- 查看次数
    correct_count   INT DEFAULT 0,              -- 正确次数（游戏/测验）
    wrong_count     INT DEFAULT 0,              -- 错误次数

    -- 时间
    first_learned_at TIMESTAMP NULL,            -- 首次学习时间
    last_reviewed_at TIMESTAMP NULL,            -- 最后复习时间

    -- 艾宾浩斯遗忘曲线复习标记
    next_review_at  TIMESTAMP NULL,             -- 下次复习时间
    review_stage    TINYINT DEFAULT 0,          -- 复习阶段 0-7

    created_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,

    UNIQUE KEY uk_child_char (child_id, char),
    FOREIGN KEY (child_id) REFERENCES child_profiles(id) ON DELETE CASCADE
);
```

```sql
CREATE TABLE achievements (
```

## 5. 拍绘本AI讲解功能（V2.5规划）

### 5.1 功能概述

允许家长/孩子拍摄绘本页面，AI识别文字和图像内容，生成生动的讲解音频。

### 5.2 用户流程

```
[首页]
    │
    ├── 点击"绘本模式"
    │       │
    │       ▼
    │   [绘本扫描页面]
    │       ├── 相机取景框（带边框引导）
    │       ├── 提示："请将绘本页面对准框内"
    │       └── 拍照按钮
    │
    ├── 拍照
    │       │
    │       ▼
    │   [AI处理中]
    │       ├── OCR识别文字
    │       ├── 图像理解
    │       └── 生成讲解脚本
    │
    └── 完成
            │
            ▼
        [绘本阅读页]
            ├── 原图展示
            ├── 高亮当前朗读文字
            ├── 播放/暂停/进度控制
            ├── 翻页按钮（上一页/下一页）
            └── 收藏绘本按钮
```

### 5.3 技术方案

#### 5.3.1 OCR + 视觉理解

```python
# 输入：绘本页面照片
# 处理流程：

1. OCR识别文字
   - 使用：PaddleOCR / 阿里云读光OCR
   - 输出：文字内容 + 位置坐标

2. 图像理解
   - 使用：qwen-vl-max
   - Prompt: """
     这是一本儿童绘本的页面。
     OCR识别到的文字是：{ocr_text}

     请：
     1. 理解故事内容
     2. 识别画面中的角色和场景
     3. 生成适合3-6岁儿童的讲解文案
     4. 标注每句话对应的文字位置（用于高亮）

     输出JSON格式：
```

## 6. 个性化识字游戏（V3.0规划）

### 6.1 游戏类型设计

#### 6.1.1 识字连连看

- **玩法**：将汉字和图片配对
- **适龄**：3-4岁初学者
- **难度梯度**：
  - L1: 2对（4个选项）
  - L2: 4对（8个选项）
  - L3: 6对（12个选项）+ 干扰项

#### 6.1.2 听音选字

- **玩法**：播放读音，从多个汉字中选出正确的
- **适龄**：4-5岁
- **个性化**：优先选择孩子已学但掌握度低的字

#### 6.1.3 组词大师

- **玩法**：拖拽汉字组成词语
- **适龄**：5-6岁
- **例子**："小" + "猫" = "小猫"

#### 6.1.4 汉字拼图

- **玩法**：将汉字拆分的部首拼成完整汉字
- **适龄**：5-6岁
- **教育价值**：理解汉字结构

### 6.2 个性化算法

```python
def recommend_game_chars(child_id, limit=10):
    """
    基于艾宾浩斯遗忘曲线和孩子掌握度，推荐游戏中要练习的字
    """

    # 1. 获取该孩子所有汉字掌握情况
    mastery = get_char_mastery(child_id)

    # 2. 分类
    to_review = []  # 需要复习的（根据next_review_at）
    weak = []       # 薄弱字（wrong_count > correct_count）
    new_chars = []  # 新字（mastery_level=0）

    for char in mastery:
        if char.next_review_at <= now():
            to_review.append(char)
        elif char.wrong_count > char.correct_count:
            weak.append(char)
        elif char.mastery_level == 0:
            new_chars.append(char)

    # 3. 优先级：复习 > 薄弱 > 新字
    result = (to_review[:5] + weak[:3] + new_chars[:2])

    return result
```

### 7.2 数据库选型

## 9. 总结与建议

### 9.1 关键决策建议

1. **账户模式**：采用"家长账户 + 儿童档案"模式，一个家长可管理多个孩子
